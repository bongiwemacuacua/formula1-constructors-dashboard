# Formula 1 Constructors Championship Dashboard

## Overview

This project is an interactive Power BI dashboard that analyzes the Formula 1 Constructors' Championship. It provides a clear view of team performance, championship standings, and points gained throughout the season using interactive visualizations.

## Objectives

The dashboard was built to:

- Monitor constructor performance across the season.
- Compare current points with starting points.
- Identify the highest-performing teams.
- Present Formula 1 data in an interactive and easy-to-understand format.

## Dashboard Features

- Interactive KPI cards
- Constructors' Championship standings
- Team points comparison
- Starting Points vs Current Points analysis
- Team slicer for dynamic filtering
- Custom Formula 1 themed design

## Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

## Files

- `Formula1Dashboard.pbix` – Power BI report
- `Formula1Dashboard.pdf` – Dashboard export
- `dashboard.png` – Dashboard preview

## Key Insights

- McLaren leads the Constructors' Championship with the highest points.
- The dashboard allows quick comparison of team performance throughout the season.
- Interactive filtering makes it easy to analyze individual teams.
- KPI cards provide a high-level summary of championship performance.
